API tutorial. Get up and running.

MongoDB

RESTHeart is the leading Web API for MongoDB.

Just start it and the REST API is ready to use, opening the data to the Web.

Web and Mobile applications can directly use the database via REST HTTP API calls: check our AngularJs notes example application.

Microservice

Lightweight: pipeline architecture, ~7Mb footprint, low RAM usage, starts in ~1 sec.

High throughput: see the performance tests;

Horizontally scalable: with Stateless Architecture and support for MongoDB replica sets and shards.

Zero dev time

No server side development is required for your web and mobile applications.

The Setup is simple with convention over configuration approach; Docker Container and Vagrant Box are available.

Access Control and Schema Check are provided out of the box.
